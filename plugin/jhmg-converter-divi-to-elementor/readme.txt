Convert Divi layouts into Elementor pages — no manual rebuilding required.

== Description ==

Upload a Divi layout export and get a fully converted Elementor page, ready to edit.

== Installation ==

1. Upload the plugin folder to `/wp-content/plugins/`
2. Activate via Plugins > Installed Plugins
3. Go to Tools > Divi → Elementor
